F1-Inspired SvelteKit 5 + Tailwind 4 for elite devs.

## Features

- **New Project**: Create a fresh FAF-ready project from template
- **GitHub Integration**: Add FAF to any existing repository
- **Direct URL**: Connect Grok directly via MCP server URL

## Tech Stack

- SvelteKit 5 with Svelte 5 runes ($state, $props, $effect)
- Tailwind CSS 4 with @theme
- TypeScript strict mode
- Vercel deployment

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Type Check

```bash
npm run check
```

## Performance

- Client bundle: ~14 kB (5.3 kB gzip)
- CSS: ~21 kB (4.5 kB gzip)
- Build time: <5s
